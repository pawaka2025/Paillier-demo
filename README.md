**Paillier Homomorphic Encryption Prototype (Python)**

Implemented a modular from-scratch Paillier cryptosystem including key generation, encryption/decryption, and homomorphic addition and scalar multiplication over modular arithmetic.

Demonstrated correctness via end-to-end validation against plaintext computation, including signed integer handling and structured parameter design.
